# Antagonistas — Instrucciones

## ¿Cómo crear un nuevo antagonista?

1. Copia la carpeta `_template/`
2. Renómbrala con el nombre del personaje
3. Define la dimensionalidad al inicio de cada archivo: `( ) 3D  ( ) 2D`
4. Llena solo las secciones que correspondan según la dimensionalidad

---

## Dimensionalidad del antagonista

**3D** — El antagonista tiene psicología completa, motivación profunda, secreto, hábitos y posible arco. Cree genuinamente que está haciendo lo correcto. Es un espejo del protagonista.

**2D** — El antagonista es una fuerza, un arquetipo o un obstáculo. No necesita justificación interna elaborada. Puede ser igual de efectivo si la historia lo requiere así.

La decisión entre 3D y 2D no es de calidad — es de función dramática.

---

## Orden de llenado

**Si es 3D:**
```
1. Ficha.md
2. Psicologia.md
3. Motivacion.md
4. Arco.md
5. Ejercicio_Escena.md
```

**Si es 2D:**
```
1. Ficha.md        ← solo secciones [2D · 3D]
2. Motivacion.md   ← solo secciones [2D · 3D]
```

---

## Archivo por archivo

**`Ficha.md`** — Igual que el protagonista, pero con una pregunta adicional clave: ¿por qué cree que tiene razón? El antagonista que no tiene respuesta a eso es plano por default.

> **Tip 3D:** La sección "Historia personal relevante" es donde se construye el espejo con el protagonista. Busca el origen o la herida que comparten — y qué elección diferente los separó.

**`Psicologia.md`** *(solo 3D)* — El antagonista 3D tiene los mismos mecanismos internos que el protagonista: miedos, mentiras que se cree, momentos de humanidad. El archivo incluye una sección específica sobre su parecido y diferencia con el protagonista.

> **Tip:** El "momento de duda o humanidad" es donde el espectador decide si puede entender al antagonista aunque no lo justifique. Ese es el objetivo.

**`Motivacion.md`** — Las secciones [2D·3D] cubren lo esencial: qué quiere, por qué choca con el protagonista, cuál es su plan. Las secciones [3D] añaden la necesidad emocional profunda y el análisis de Maslow.

**`Arco.md`** *(solo 3D)* — No todos los antagonistas 3D tienen arco, pero si lo tienen la historia gana profundidad. El archivo empieza con una pregunta directa: ¿tiene arco o no?

**`Ejercicio_Escena.md`** *(solo 3D)* — El reto aquí es más difícil que con el protagonista: el antagonista debe parecer razonable o incluso simpático en la escena. Si logras eso, el personaje funciona.
