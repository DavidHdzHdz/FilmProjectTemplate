# Guion Técnico

> El guion técnico integra el guion literario con las indicaciones técnicas de cámara, iluminación y sonido. Es el documento que usa el equipo en el set.

---

## Cómo usar este documento

Cada escena incluye:
- El texto del guion
- Las indicaciones de cámara
- Las notas de arte / locación
- Las notas de sonido

---
