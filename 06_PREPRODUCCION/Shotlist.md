# Shotlist (Lista de Planos)

> La shotlist es la lista técnica completa de todos los planos a rodar, ordenados por escena. Es la hoja de ruta del rodaje.

---

## Cómo leer esta lista

| Campo | Descripción |
|---|---|
| # | Número de plano |
| Escena | Número de escena del guion |
| Tipo | Tipo de plano (PG, PM, PP, etc.) |
| Movimiento | Movimiento de cámara |
| Lente | Focal estimada |
| Descripción | Qué muestra el plano |
| Personajes | Quién aparece |
| Duración est. | Duración estimada en pantalla |
| Notas | Instrucciones especiales |
