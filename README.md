# Film Project Template

Framework de desarrollo cinematográfico en Markdown, organizado en 9 fases desde la idea inicial hasta la postproducción. Basado en la metodología de *Escribir Cine* de Alexander Steele.

Diseñado para ser usado solo o en equipo — cineastas independientes, estudiantes de cine y equipos de producción.

---

## Estructura

```
FilmProjectTemplate/
├── 01_CONCEPTO/          ← Empieza aquí: premisa, logline, sinopsis
├── 00_PROJECT_OVERVIEW/  ← Visión general y cronograma (después del concepto)
├── 02_MUNDO/             ← Universo narrativo, reglas, lugares
├── 03_PERSONAJES/        ← Protagonistas, antagonistas, secundarios, terciarios
├── 04_ESCRITURA/         ← Del brainstorm al guion final
├── 05_VISUAL/            ← Referencias visuales, paleta, fotografía
├── 06_PREPRODUCCION/     ← Storyboard, shotlist, plan de rodaje
├── 07_PRODUCCION/        ← Diario de rodaje
└── 08_POST/              ← Edición, color, sonido, música
```

---

## Orden de llenado

```
01_CONCEPTO → 00_PROJECT_OVERVIEW → 02_MUNDO + 03_PERSONAJES
→ 04_ESCRITURA + 05_VISUAL → 06_PREPRODUCCION → 07_PRODUCCION → 08_POST
```

Cada carpeta tiene su propio `INSTRUCCIONES.md` con el orden específico y tips de uso.

---

## Cómo usar este template

1. Clona o descarga el repositorio
2. Lee `INSTRUCCIONES.md` en la raíz del proyecto
3. Empieza por `01_CONCEPTO/` — sigue el orden del `INSTRUCCIONES.md` de cada carpeta
4. Para los personajes: copia la carpeta `_template/` dentro del tipo correspondiente y renómbrala con el nombre del personaje

## Sistema de personajes

Los personajes se organizan por tipo y dimensionalidad dramática:

| Tipo | Dimensionalidad | Carpeta |
|---|---|---|
| Protagonistas | Siempre 3D | `03_PERSONAJES/Protagonistas/` |
| Antagonistas | 3D o 2D | `03_PERSONAJES/Antagonistas/` |
| Secundarios | 3D o 2D | `03_PERSONAJES/Secundarios/` |
| Terciarios | 2D o 1D | `03_PERSONAJES/Terciarios/` |

Cada tipo tiene una carpeta `_template/` con los archivos en blanco y carpetas de ejemplo del universo **Bad Request / Return Void** como referencia de llenado.

## Metodología

Este framework aplica un enfoque fijo e intencional — no está pensado para ser personalizado, sino para ser seguido. Las herramientas de desarrollo de personajes incluyen:

- Motivación en dos niveles (quiere vs. necesita)
- Pirámide de Maslow aplicada al arco dramático
- El secreto que contradice la imagen pública
- Hábitos filmables que revelan psicología sin diálogo
- La mentira que el personaje se cree (núcleo del arco)
- Ejercicio de escena implícita
