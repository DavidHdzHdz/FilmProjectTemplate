# Diario de Rodaje

> El diario de rodaje es el registro diario del set. Captura lo que ocurrió, lo que cambió y lo que necesitas resolver. Es invaluable para la postproducción y para futuros proyectos.

---

## Cómo usar este diario

- Completa cada entrada al final del día de rodaje.
- Sé honesto/a: el diario es para ti, no para el público.
- Registra los problemas para poder resolverlos al día siguiente.
- Guarda las ideas que surgieron en el set.
