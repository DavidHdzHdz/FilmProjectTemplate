# Cinema Project Template

## Bienvenido a tu proyecto cinematográfico

Esta carpeta es el punto de entrada de tu proyecto. Aquí encontrarás toda la estructura necesaria para desarrollar tu historia desde la idea inicial hasta la postproducción.

---

## Cómo usar esta plantilla

Cada carpeta representa una fase del desarrollo:

| Carpeta | Contenido |
|---|---|
| `00_PROJECT_OVERVIEW` | Visión general, hoja de ruta y plazos |
| `01_CONCEPTO` | La idea: premisa, logline, sinopsis, tema |
| `02_MUNDO` | El universo narrativo donde ocurre la historia |
| `03_PERSONAJES` | Fichas completas de cada personaje |
| `04_ESCRITURA` | Del brainstorm al guion final |
| `05_VISUAL` | Lenguaje visual, referencias, paleta de color |
| `06_PREPRODUCCION` | Storyboard, shotlist, plan de rodaje |
| `07_PRODUCCION` | Diario de rodaje y notas en set |
| `08_POST` | Edición, sonido, música y color |
