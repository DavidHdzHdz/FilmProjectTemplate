# Referencias Visuales

> Documento de trabajo para organizar tus referencias visuales. Los archivos de imagen van en la carpeta `Referencias/`.

---

## Cómo usar este documento

Para cada referencia, anota:
- **Fuente:** película, fotógrafo, pintor, etc.
- **¿Qué tomas de ella?** No solo "me gusta", sino qué específicamente te sirve.
- **¿Dónde aplica?** ¿A toda la película, a una escena, a un personaje?
